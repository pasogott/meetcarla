# Changelog

All notable changes to this project will be documented in this file.

## 0.1.0 - 2026-03-13

Initial public release of Carla.

### Added

- local-first Tauri desktop app shell with tray integration and dedicated windows
- macOS-native meeting capture via bundled Swift helper
- on-device transcription pipeline with MLX primary and Whisper fallback
- SQLite-backed meeting storage, transcript search, playback state, and exports
- AI summaries, task extraction, scratchpad editing, and Ask AI workflows
- speaker identification, templates, tags, Notion export, and webhooks
- GitHub Actions release workflow that rebuilds the bundled Python runtime and uploads a macOS DMG to GitHub Releases
